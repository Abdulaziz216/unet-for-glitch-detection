# Directory Overview

- **`context.py`**  
  Bootstraps the project’s `src/` import path.

- **`generationScript.ipynb`**  
  Main notebook: runs the TOD→image/mask pipeline and writes outputs.

- **`tod_simulator_v2.ipynb`**  
  Legacy notebook with original TOD simulation experiments.

- **`README.md`**  
  This document.

- **`__pycache__/`**  
  Python bytecode cache (ignore).

> **Tip:** The output directory is determined by **`DATASET_NAME`** in Cell 3 of `generationScript.ipynb`—modify it as needed for each run.
