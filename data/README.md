## Contents & Formats (simulated sample)

The sample folder may include:

* `images/` â€” Long-exposure focal-plane images (`*.npy`, `*.npz`, or `*.png`).
* `masks/` â€” Semantic segmentation masks with classes: **background / Point Sources (PS) / Cosmic Rays (CR)**.
* `tod/` â€” Optional raw or sliced timestream arrays used to form each image.
* `meta.*` â€” Optional metadata (JSON/CSV) with generation parameters, seeds, and file mappings.

Files across `images/`, `masks/`, and `tod` share a common stem (e.g., `sample_000123.*`) for 1:1 matching.
